# Progress Notes

Use this directory for session plans, closeouts, and decision records.

## Naming

Use ISO dates and short task names:

```text
YYYY-MM-DD-[task-name]-plan.md
YYYY-MM-DD-[task-name]-closeout.md
YYYY-MM-DD-[task-name]-decision.md
```

Examples:

```text
2026-05-05-ai-collaboration-rules-closeout.md
2026-05-05-local-treesitter-parser-closeout.md
```

## Required Closeout Content

Each closeout should include:

- summary
- files or local paths changed
- validation run
- result
- follow-ups or known risks

## Rule

`AGENTS.md` is stable rules, not project history.
`docs/ai/current-reality.md` is current project state.
`progress/` is historical session record.
